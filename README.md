#### installing in termux :
  + `apt update -y`
  + `apt upgrade -y`
  + `git clone https://github.com/C4ssif3r/termux-root/`
  + `cd termux-root`
  + `chmod 777 main`
  - or
  + `chmod +x main`
  + `./main`
